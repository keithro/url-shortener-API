URL Shortener Microservice API

Coded by Keith Rodriguez

My project for the URL Shortener Microservice challenge as part of the curriculum for the Back End Development Certification on Free Code Camp.

View the source code for this project.

Example Usage

Create Shortened Url:

https://UPDATE_ME_AFTER_DEPLOY.herokuapp.com/new/https://www.yourlongurl.com/wish-this-could-be-shorter

Response Output:

{ originalUrl: "https://www.yourlongurl.com/wish-this-could-be-shorter", shortenedUrl: "http://UPDATE_ME_AFTER_DEPLOY.herokuapp.com/3206" }

Navagation to Short Url:

http://UPDATE_ME_AFTER_DEPLOY.herokuapp.com/3206

Will Redirect To:

https://www.yourlongurl.com/wish-this-could-be-shorter
